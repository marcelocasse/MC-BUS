
# MC BUS

MC BUS brinda la solución de saber los horarios de colectivos en las zonas de Liebig, San José y Colón Entre Rios, Argentina.
## Tech Stack

**Client:** HTML, CSS (SASS), Javascript, Bootstrap

**Server:** Django, SQLITE3.
